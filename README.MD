# Safe Exam Browser & Config Tool

## Описание проекта

> [!WARNING]
> ### Цель
> Этот проект создан исключительно в учебных целях.
> 
> ### Предостережение
> Любое несанкционированное использование этого подхода для экзаменов, а также попытки взлома без согласия автора экзамена считаются незаконными и противоречат принципам этичного поведения.
>
> ### Авторство
> Исходный код приложения и права на него принадлежат автору - [
Safe Exam Browser](https://github.com/SafeExamBrowser).

> так как исходный проект распространяется под лицензией MPL-2.0: я публикую только измененные файлы, а не весь проект, исходный код вы можете найти в разделе `Авторство` по ссылке.

## Технологии

Язык программирования: [C#](https://learn.microsoft.com/ru-ru/dotnet/csharp/) \
Платформа: [.NET](https://dotnet.microsoft.com/)

#### Safe Exam Browser
Всегда запускается в режиме `kiosk mode = None`, что позволяет пользоваться функциями компьютера. Браузер позволяет пользоваться буфером обмена, а также делать скриншоты экрана. Браузер будет спокойно запускаться в виртуальной машине.

#### SEB Config Tool
> Не вносите изменения, если сомневаетесь, оно и так заработает

Всегда сохраняется с одним и тем же `BrowserExamKey` и `ConfigurationKey`, что позволяет настраивать файл настроек (config) под себя.
 - BrowserExamKey на 622 строчке кода в файле `SebWindowsConfig\Utilities\SEBProtectionController.cs`
 - ConfigurationKey на 668 строчке кода в файле `SebWindowsConfig\Utilities\SEBProtectionController.cs`

## Как использовать
1. Скачайте и установите исходный Safe Exam Browser версии 3.8.0 по ссылке:
[Для Windows](https://github.com/SafeExamBrowser/seb-win-refactoring) \
Для других операционных систем взлом не производился 

2. Скачайте последнюю версию проекта в разделе `releases` и замените исходные файлы браузера на скачанные.
Примерные пути для Safe Exam Browser: "C:\Program Files\SafeExamBrowser\Application", для SEB Config Tool: "C:\Program Files\SafeExamBrowser\Configuration".

> Еще раз, заменить только повторяющиеся файлы, другие не трогать!

Используйте Safe Exam Browser с любым файлом настроек и наслаждайтесь!
